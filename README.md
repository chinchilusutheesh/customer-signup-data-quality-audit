# Customer Sign-Up Behaviour & Data Quality Audit
## Project Overview
This project analyzes customer sign-up data for a SaaS company to evaluate data quality and uncover business insights related to user acquisition, demographics, and marketing behavior.
The objective is to identify missing or inconsistent data, clean and standardize the dataset, and provide actionable insights to support marketing and onboarding decisions.
## Key Objectives
- Perform data quality assessment and cleaning.
- Standardize inconsistent categorical values.
- Handle missing values and duplicates.
- Generate summary statistics and aggregations.
- Answer business questions related to sign-up trends and customer behavior.
## Dataset
The dataset includes customer sign-up information such as:
- Signup date
- Acquisition source
- Region
- Selected subscription plan
- Marketing opt-in status
- Demographics (age, gender)
> Note: The dataset may not be publicly included due to academic restrictions.
## Tools Used
- Python  
- Google Colab  
- Pandas, NumPy  
- Matplotlib, Seaborn  
## Analysis Highlights
- Identified missing and inconsistent values across multiple columns.
- Standardized categorical fields such as plan type and gender.
- Removed duplicate customer records.
- Generated weekly sign-up trends and source distribution summaries.
- Analyzed marketing opt-in behavior across demographics.
## Business Value
This analysis supports business teams by improving data reliability and highlighting trends in customer acquisition channels, demographics, and plan selection.
